# Nexara Blockchain

Next-generation Layer-1 blockchain built from scratch in Rust with adaptive sharding, a custom virtual machine (NXVM), a smart contract language (NexLang), and cross-chain bridging.

## Features

- Custom virtual machine (NXVM) for smart contract execution
- NexLang: purpose-built smart contract language
- Adaptive sharding for horizontal scalability
- Cross-chain bridge for asset transfers
- BFT-based consensus mechanism
- Built-in tokenomics and governance modules

## Tech Stack

Rust, Custom VM, NexLang, Networking, Cryptography

## Getting Started

### Prerequisites

- Rust toolchain
- cargo

### Installation

```bash
git clone https://github.com/karthik-idikuda/crypto.git
cd crypto
cargo build --release
```

### Usage

```bash
cargo run --bin nexara-node
```
